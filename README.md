# Frontend Book Finder

Book Finder is a web application for users to search and find books of a subject they want to know about, or a specific genre, author, etc.

## Overview

This is the frontend code, built with ReactJS and using Material-UI library. I worked on this as a way of gaining experience working with React, and especially with Golang, which is the technology I use in the backend. Since my focus was the backend development, the frontend is meant to be simple.
